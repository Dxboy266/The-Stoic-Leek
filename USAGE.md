# 使用指南

## 快速开始

### 1. 获取 API 密钥

1. 访问 [硅基流动官网](https://siliconflow.cn)
2. 注册账号（支持微信、手机号等方式）
3. 进入控制台，创建 API 密钥
4. 复制 API 密钥（格式类似：`sk-xxxxxxxxxxxxxx`）

**注意**：硅基流动提供免费额度，足够个人使用。

### 2. 配置应用

1. 打开应用后，在左侧边栏找到"API 密钥设置"
2. 将复制的 API 密钥粘贴到输入框中
3. 密钥会自动保存在当前会话中

### 3. 使用应用

#### 输入盈亏金额

- **盈利**：输入正数，如 `100`（表示盈利 100 元）
- **亏损**：输入负数，如 `-100`（表示亏损 100 元）
- **持平**：输入 `0`

#### 生成处方

1. 输入金额后，点击"🎯 生成处方"按钮
2. 等待 AI 生成（通常 2-5 秒）
3. 查看生成的处方，包括：
   - 今日盈亏状态
   - 健身任务（动作和数量）
   - AI 生成的个性化建议

#### 重新生成

如果对当前处方不满意：
1. 点击"🔄 重新生成"按钮
2. 系统会使用相同的金额生成新的处方
3. 健身动作和 AI 建议都会改变

## 使用场景

### 场景 1：亏损后的情绪管理

**输入**：`-500`（亏损 500 元）

**系统反应**：
- 心情判断：焦虑
- 健身任务：50 个深蹲/俯卧撑/卷腹（随机）
- AI 建议：幽默嘲讽 + 斯多葛哲学安慰

**目的**：通过运动发泄负面情绪，用哲学智慧重获理性

### 场景 2：盈利后的冷静提醒

**输入**：`1000`（盈利 1000 元）

**系统反应**：
- 心情判断：兴奋
- 健身任务：50 个波比跳/开合跳（随机）
- AI 建议：打击嚣张 + 风险警示

**目的**：通过运动冷静头脑，避免过度兴奋导致冲动决策

### 场景 3：持平时的平常心

**输入**：`0`（持平）

**系统反应**：
- 心情判断：平淡
- 健身任务：30 秒平板支撑/拉伸（随机）
- AI 建议：平常心鼓励

**目的**：保持身心平衡，继续保持理性和耐心

## 健身任务计算规则

### 亏损场景
- **公式**：abs(亏损金额) ÷ 10
- **最小值**：1 个动作
- **动作池**：深蹲、俯卧撑、卷腹、高抬腿
- **示例**：
  - 亏损 50 元 → 5 个动作
  - 亏损 5 元 → 1 个动作（最小值）
  - 亏损 1000 元 → 100 个动作

### 盈利场景
- **公式**：盈利金额 ÷ 20
- **最小值**：1 个动作
- **动作池**：波比跳、深蹲、俯卧撑、开合跳
- **示例**：
  - 盈利 100 元 → 5 个动作
  - 盈利 10 元 → 1 个动作（最小值）
  - 盈利 2000 元 → 100 个动作

### 持平场景
- **固定值**：30 秒
- **动作池**：平板支撑、拉伸、靠墙静蹲

## 常见问题

### Q: API 密钥会被保存吗？
A: API 密钥只保存在当前浏览器会话中，关闭浏览器后需要重新输入。我们不会存储您的密钥。

### Q: 为什么生成失败？
A: 可能的原因：
1. API 密钥无效或过期
2. 网络连接问题
3. API 服务暂时不可用
4. 请求过于频繁

**解决方法**：检查密钥、稍后重试、或联系硅基流动客服。

### Q: 可以修改健身动作吗？
A: 当前版本动作是随机选择的，您可以通过"重新生成"来获取不同的动作。

### Q: 手机上可以使用吗？
A: 完全可以！应用已针对移动设备优化，在手机浏览器中访问即可。

### Q: 数据会被记录吗？
A: 不会。所有数据只在当前会话中存在，不会被上传或存储。
